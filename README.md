# Naive OS :cat:

## About
a naive OS kernel which only support automate processes switch. It is written in C and risc-v

## Features
- [x] Kernel Mode and User mode
- [x] Virtual Memory Management
- [x] page fault process
- [x] multi-processes
- [x] fork

## Usage
```bash
# run
$ make run
# debug
$ make debug
# clean
$ make clean
```
